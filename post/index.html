Mattermost****的概述 Mattermost是一个开源平台，用于跨团队和工具的安全通信，协作和工作编排。 Mattermost专为软件开发和工程用例而构建，并与丰富的第三方开发人员工具生态系统深度集成。Mattermost使公司能够完全控制其数据;通过自托管和私有云部署选项以及对源代码的访问，开发人员可以直接为专为他们构建的共享、灵活且可扩展的平台做出贡献。 我们支持关键的开发人员协作用例： 敏捷软件开发，面向构建和操作软件的开发人员，以提供更高的生产力。技术特性和与开发人员使用的工具的深度集成支持敏捷软件开发的所有阶段，包括规划、构建、发布和操作。 开发人员工作流编排，适用于认为软件开发至关重要的公司的高级技术角色和工程/IT 管理。我们通过先进的开发人员工作流程以及与任务关键型应用程序的大规模集成，帮助他们成功运营数字化企业。 面向开发人员团队的事件解决方案，用于响应时间关键型事件，通过持续的流程改进提供更快的解决时间，减少容易出错的事件响应。 在需要最高级别的安全性、隐私性、合规性和数据控制的公司中，为 InfoSec 和 CIO/CISO 提供安全协作。我们提供内部部署的自托管协作解决方案，该解决方案完全安全，高度可定制，并通过严格的访问控制进行内部管理，并与SaaS无法实现的内部安全/合规性框架轻松集成。 Mattermost****的平台 Mattermost是生产力工具的集成平台，可提供工作流编排并消除上下文之间的切换。Mattermost提供的解决方案结合了基于渠道的沟通、结构化的工作流程执行和项目工作流程。该平台提供单一安全部署、共享用户管理和通用用户体验，并具有支持企业级协作的全面工具集。 该平台在设计上是完全可扩展的，支持丰富的第三方应用程序和集成生态系统。Mattermost使用开放式API，开发人员框架，开源自定义和社区驱动的增强功能轻松增强和自定义。Mattermost还支持20种语言，以满足全球团队的需求。 Mattermost****解决的协作工作流程 Mattermost平台集成了三个主要工具，涵盖了广泛的协作用例。每个工具都专注于一个主要工作流程，可以无缝组合使用，以实现实时协作以及持续时间从几天到几个月不等的流程和项目。 使用渠道保持联系并支持您的通信工作流程 基于渠道的通信工作流需要实时信息共享、灵活性和临时活动，其中通信工作流的生命周期通常以秒到小时为单位进行测量。 通过频道，您可以使用无限制且完全可搜索的消息历史记录来组织团队的实时 1：1 和群组通信。共享文件、图像、链接等材料;用表情符号和GIF分享情绪;并通过丰富的 Markdown 格式、代码语法突出显示和多语言支持来表达想法。通过集成的语音/视频会议促进面对面的连接。渠道是 Bug 分类、每日站立会议、发布公告和团队对话的理想选择。![image-20230311202721685](/Users/lvchenyu/Library/Application Support/typora-user-images/image-20230311202721685.png) 使用行动手册来支持您规定的工作流程并信任流程 结构化工作流是需要可靠且可重复的过程的特定方案和用例。这些用例非常适合共享清单、自动化和回顾，其中生命周期以小时到周为单位进行衡量。一些示例包括事件协作和软件版本。 行动手册简化了复杂的重复性流程，使团队能够实现特定且可预测的结果。使用内置清单、触发器、自动化、工具集成和回顾来编排规定的工作流程。通过状态更新和累积统计数据提高利益相关者的可见性。自定义行动手册以符合组织实践，并动态调整以在高度动态的情况下快速响应。行动手册在事件协作中至关重要，以避免和最大限度地减少代价高昂的业务中断，以及在按时交付高质量软件的软件发布流程中至关重要。![image-20230311202756957](/Users/lvchenyu/Library/Application Support/typora-user-images/image-20230311202756957.png) 使用开发板实现您的里程碑并支持您的项目工作流 项目工作流包括推动里程碑实现的用例，例如项目规划、执行和任务管理，在这些用例中，团队需要在一组共享的任务、里程碑和目标上保持一致。项目工作流的典型生命周期以数周到数月为单位进行度量。项目工作流可以包括各种范围，从引入新产品或功能到异地规划团队。 Boards 是一种可交付成果和任务管理解决方案，可帮助团队使用熟悉的看板结构实现项目里程碑。协调整个组织的工作，并将时间和精力投入到最重要的事情上。通过明确定义的任务、所有者、清单和截止日期，按计划进行。提高透明度并保持所有资源可用，包括文档、图像和链接。此外，开发板非常适合跟踪冲刺 （sprint） 的任务和路线图规划中的功能。![image-20230311202656582](/Users/lvchenyu/Library/Application Support/typora-user-images/image-20230311202656582.png)
